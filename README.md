# 项目简介

## 核心功能
- 数据采集：利用爬虫技术从爱奇艺电影网站获取电影相关信息。
- 数据存储：将采集到的数据存储到MySQL数据库中，包括电影基本信息、评分等。
- 数据清洗：对采集到的数据进行清洗和预处理，保证数据质量和一致性。
- 数据分析：分析电影评分分布、不同类型电影的数量分布、评分和演员影响力等。
- 可视化展示：基于Flask框架和Echarts图标库，将数据分析结果以图表形式展示。

## 适用场景
- 影视行业数据分析：帮助影视行业从业者了解电影市场趋势和用户评价。
- 学术研究：为电影学、传播学等学术领域提供数据支持。
- 爱奇艺用户：提供个性化推荐和深入的电影信息了解。

## 技术特点
- 开源技术栈：使用Python、Flask、MySQL、html、css、js、echarts等开源技术。
- 数据可视化：利用Echarts图表库实现数据的高效可视化展示。
- 交互式查询：用户可以通过自定义查询和过滤功能，深入挖掘感兴趣的电影信息。
